# Autorizador

## Setup

1. Crie um ambiente virtual:
    ```sh
    python3 -m venv venv
    ```

2. Ative o ambiente virtual:
    ```sh
    source venv/bin/activate
    ```

3. Instale as dependências:
    ```sh
    pip install -r requirements.txt
    ```

## Execução

Para executar a aplicação:
```sh
./run.sh
